# PassOP 

**PassOP** is a sleek, minimal password manager built with **React**, **Tailwind CSS**, and **UUID**. It allows users to securely **save**, **edit**, **delete**, and **search** passwords â€” all stored locally in the browser using `localStorage`. No backend needed, 100% privacy-focused.

##  Features

-  Save and manage passwords (locally stored)
-  Edit and update credentials inline
-  Instantly delete entries
-  Copy password to clipboard with one click
-  Real-time search/filter for saved passwords
-  Unique IDs for each entry using `uuid`
-  Fully responsive UI with Tailwind CSS
-  Smooth, modern animations for better UX
